# Supported Versions

The following table lists the versions of `sera-data` that are currently supported with security updates:

| Version | Supported | Security Fixes |
|---------|-----------|----------------|
| 1.x.x   | Yes       | Yes            |
| < 1.0.0 | No        | No             |

## Node.js & Dependency Compatibility

- **Node.js**: `= 22.x` (LTS runtime)
- **pnpm**: `>= 9.4.0`
- **PostgreSQL**: `>= 15.0`
- **Viem**: `^2.16.5`
- **Fastify**: `^4.28.0`
- **Kysely**: `^0.27.3`
